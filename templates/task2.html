Network Builder task two - Network Addresses

Computers are a lot like houses - they have addresses which let us know where to find them. What makes it more complicated is that there is more than one type of address that we use to describe computers. This challenge will look at a couple of ways that we can identify individual computers.

Work through the tasks and complete the questions, which will reveal the flag needed to complete the challenge.


Physical address

Every network card on a computer has a physical address, also known as a MAC address. This address is unique to that network interface and is used to talk to other computers within the local part of the network (known as the subnet).
